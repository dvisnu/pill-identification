# 💊 Two-Stage Pill Identification System
### YOLOv8 (Detection) + ResNet-50 (Classification)

A deep learning pipeline that identifies pills in two stages:
1. **Stage 1 — Detection**: YOLOv8 nano detects and localizes pills in an image, cropping them out.
2. **Stage 2 — Classification**: A fine-tuned ResNet-50 classifies each cropped pill by name/color.

## 🧠 Model Architecture

### Stage 1 — YOLOv8 Nano (Object Detection)
- Pretrained on COCO, fine-tuned on custom pill dataset
- Input size: `416 × 416`
- Epochs: `50`, Batch size: `16`
- Detects 8 pill classes and crops bounding boxes for Stage 2

### Stage 2 — ResNet-50 (Image Classification)
- Pretrained on ImageNet, base layers frozen
- Custom FC head: `Linear(2048→512) → ReLU → Dropout(0.3) → Linear(512→7)`
- Epochs: `10`, Batch size: `32`
- Classifies each cropped pill into one of 7 named/color categories
- Overall validation accuracy: **87%**

## 🛠️ Setup & Usage

### 1. Clone the Repository
```bash
git clone https://github.com/dvisnu/pill-identification.git
cd pill-identification
```

### 2. Install Dependencies
```bash
pip install -r requirements.txt
```

### 3. Google Drive Folder Structure
Both notebooks read from Google Drive. Your Drive should be organised as:
```
Pill_identification/
├── prototype_dataset/                     # YOLO dataset (train/valid/test splits + data.yaml)
├── cropped_pills/                         # Auto-generated by train.ipynb (Stage 1 → Stage 2 bridge)
├── outputs_results                        # YOLO training outputs & weights (auto-generated)
├── pill_classifier_resnet50_best.pth      # ResNet weights
└── pill_detailed_contents.csv             # Pill metadata CSV
```
You can download the dataset from : [dataset](https://drive.google.com/drive/folders/1fh16IARx_1ItnNHl0Epc8E0pcjN45evX?usp=sharing)

### 4. Run Training — `train.ipynb`
Open `notebooks/train.ipynb` in [Google Colab](https://colab.research.google.com/) and run top-to-bottom.

| Step | What happens |
|------|-------------|
| Setup | Mounts Drive, installs packages, sets paths |
| Stage 1 | Trains YOLOv8n for 50 epochs on the pill detection dataset |
| Cropping | Runs YOLO on all splits → saves cropped pills to `cropped_pills/` |
| Stage 2 | Trains ResNet-50 classifier on the cropped pills for 10 epochs |
| Output | Saves `pill_classifier_resnet50_best.pth` to `BASE_DIR` in Drive |

> Update `BASE_DIR` in the paths cell if your Drive folder is in a different location:
> ```python
> BASE_DIR = '/content/drive/My Drive/YOUR_FOLDER/Pill_identification'
> ```

### 5. Run Evaluation — `evaluate.ipynb`
Open `notebooks/evaluate.ipynb` in Colab **after** `train.ipynb` has completed.

| Step | What happens |
|------|-------------|
| Setup | Mounts Drive, installs packages, sets paths |
| Stage 1 Eval | YOLO `.val()` on test set — per-class mAP, precision, recall |
| Visual Check | Saves annotated detection images to `/content/runs/detect/predict` |
| Stage 2 Eval | Loads best ResNet-50 weights, runs on validation set |
| Metrics | Raw + normalized confusion matrix, classification report |
| Inference | Full two-stage pipeline on a single test image |

> `evaluate.ipynb` is self-contained — it does **not** depend on any variables from `train.ipynb`.

## 🔧 Data Preprocessing & Augmentation
Done via **Roboflow** before training:

- **Preprocessing**: Auto-Orient, Resize (Stretch to 416×416)
- **Augmentations**: 90° rotations, Random crop (0–77% zoom), ±45° rotation, Shear ±15°, Hue/Saturation/Brightness/Exposure jitter, Blur up to 3px, Noise up to 5%
