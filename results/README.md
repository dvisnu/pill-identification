# 📊 Results

This folder contains training metrics and evaluation outputs from the **YOLOv8 Stage 1** model.  
All files are auto-generated by Ultralytics during `model.train()` and `model.val()`.

> ResNet-50 (Stage 2) confusion matrix and classification report are generated at runtime inside `evaluate.ipynb`.

---

## 📁 Files

| File | Description |
|------|-------------|
| `results.png` | Training curves - loss, precision, recall, mAP over 50 epochs |
| `results.csv` | Raw epoch-by-epoch numbers behind `results.png` |
| `BoxPR_curve.png` | Precision-Recall curve per class (mAP@0.5 = 0.983) |
| `BoxF1_curve.png` | F1-Confidence curve - best F1 = 0.82 at confidence 0.770 |
| `BoxP_curve.png` | Precision-Confidence curve - reaches 1.00 at confidence 0.969 |
| `BoxR_curve.png` | Recall-Confidence curve - max recall 0.99 at low confidence |
| `confusion_matrix.png` | Raw confusion matrix on validation set |
| `confusion_matrix_normalized.png` | Normalized confusion matrix (row = actual class) |
| `labels.jpg` | Dataset label distribution and bounding box statistics |

---

## 📈 Training Curves (`results.png`)

Shows how the model improved over 50 epochs:

- **Train/Val losses** (box, cls, dfl) - all steadily decrease, confirming no overfitting
- **Precision & Recall** - stabilise above 0.95 after ~epoch 20
- **mAP@50** - converges near 0.995 (near-perfect detection)
- **mAP@50-95** - reaches ~0.92, showing strong localisation across IoU thresholds

## 🔲 Confusion Matrix (`confusion_matrix_normalized.png`)

Key observations:
- **Blue, Pink, Red, White** - perfect recall (1.00), no misclassifications
- **Ibuphil Cold 400-60** - 100% recall but some Xyzall 5mg instances get misclassified into it
- **Cipro 500** - 85% recall; some instances predicted as background (8%) or Xyzall 5mg (8%)
- **Xyzall 5mg** - weakest class at 80% recall; 37% of white-class detections are mislabelled as Xyzall 5mg
- **Ibuphil 600 mg** - no instances in the validation set (class imbalance)
